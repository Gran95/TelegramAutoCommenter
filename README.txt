Telegram AutoCommenter

1. Запустите TelegramAutoCommenter.exe
2. Укажите ссылку на пост
3. Выберите файл с комментариями (comments.txt)
4. Установите интервал между комментариями
5. Нажмите 'Старт'

Для смены аккаунта используйте кнопку 'Сменить аккаунт'.
Комментарий отправляется один раз, без повторов. Программа завершит работу, когда все шаблоны будут использованы.
